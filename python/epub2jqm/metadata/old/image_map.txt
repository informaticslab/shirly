# image_map.txt - This file describes how image links found in the EPUB file are to be mapped
# 				into the app. Each line has this format:
#				
#		"Image file path in quotes",MAP_OPTION
#		
#		The MAP_OPTION can be one of two options:
#
#			rem - this means that the image link should be removed
#					Example:	"../Images/cervicitis-rec.png",rem
#			use - this means that the image link should be used
#					Example:	"../Images/cervicitis-rec.png",use
#
#			a-b-c where a-b-c is a table or comma separated list of tables that should replace
#				  the image link. Tables should be specified in the PDF-Page, PDF-Column, PDF-table number 
#				  format that is found in the table-data.txt metadata file.
#					Example:	"../Images/cervicitis-rec.png",52-1-1,52-1-2
